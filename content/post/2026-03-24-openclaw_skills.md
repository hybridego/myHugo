# OpenClaw Skills

- apple-notes — Apple 메모 생성/검색/수정/정리
- apple-reminders — Apple 미리알림 추가/수정/완료 처리
- camsnap — RTSP/ONVIF 카메라 화면 캡처/클립 저장
- clawhub — ClawHub에서 스킬 검색/설치/업데이트/배포
- coding-agent — 코딩 작업을 Codex/Claude Code 같은 에이전트에 위임
- discord — Discord 메시지/채널 관련 작업
- gemini — Gemini CLI로 질의응답/요약/생성
- gh-issues — GitHub 이슈 가져와서 수정 작업/PR 흐름 관리
- github — GitHub CLI로 이슈/PR/액션/run 관리
- goplaces — Google Places로 장소 검색/상세정보 조회
- healthcheck — 보안 점검, 방화벽/SSH/업데이트/노출 상태 진단
- imsg — iMessage/SMS 조회/전송
- model-usage — 모델별 사용량/비용 요약
- nano-pdf — 자연어로 PDF 수정
- node-connect — OpenClaw 노드 연결/페어링 문제 진단
- obsidian — Obsidian 볼트의 메모 읽기/정리/자동화
- openai-whisper — Whisper로 로컬 음성→텍스트 변환
- peekaboo — macOS 화면 보고 앱 UI 클릭/입력/자동화
- session-logs — 이전 세션 로그 검색/분석
- skill-creator — 새 스킬 만들기, 기존 스킬 정리/개선/감사
- songsee — 오디오 스펙트로그램/특징 시각화
- sonoscli — Sonos 스피커 제어
- summarize — URL, PDF, 이미지, 오디오, 유튜브 내용 요약
- things-mac — Things 3 할 일/프로젝트 관리
- tmux — tmux 세션 원격 조작
- video-frames — 영상에서 프레임/짧은 클립 추출
- weather — 현재 날씨/예보 조회
- xurl — X(트위터) 읽기/검색/포스트/DM/API 작업
- gstack-lite — 전략, 기획, 리뷰, 제품 사고용 경량 워크플로우
- lightpanda-browser — Lightpanda 브라우저로 빠른 헤드리스 웹 작업
- self-improvement — 실패/교정/교훈 기록해서 다음에 더 잘하게 돕는 스킬
- skill-vetter — 외부 스킬 설치 전 보안 검토
- stock-analysis — 한국/미국 주식 분석 리포트 작성
